About FMF (Farmer's Market Finder)

This application was constructed using an ASP.NET Core Web API.

The API uses Entity Framework Core and an Azure SQL Database

The front-end (this site) is created using Angular 9 and Material Design Components

With this application, you're able to use either a zip code or your current location to find Farmers' Markets near you.

The list is ordered by distance, and you're able to view details about each farm when clicked on, and favorite the ones you like.

On top of storing and retrieving information about the markets in the SQL Database, all user actions are stored in the database as well, for audit and tracking reasons.

For extras, check out the Trello Board that was used to create this project.
